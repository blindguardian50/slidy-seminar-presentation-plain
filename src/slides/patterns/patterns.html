Responsive Visualization Patterns

  • “Design Patterns and Trade-Offs in Responsive Visualization for Communication” [Kim, Moritz and Hullman 2021]:
    • Targets = What is changed (data, encoding, interaction, narrative, references/layout).
    • Actions = How something is changed (recompose, rescale, transpose, reposition, compensate).
    • Pattern = Applying Action to Target to get from State A to State B.
  • Types of responsive visualization pattern:
    • Visual Patterns: Change Visual Representation.
    • Interaction Patterns: Enable User to Interact with Visualization.
    • Data Patterns: Transform Underlying Data.
