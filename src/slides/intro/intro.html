Responsive Web Design

  • Originated from need to avoid horizontal scrolling.
  • Term “Responsive Web Design” coined by Ethan Marcotte [Marcotte 2010]:
    • Flexible grids
    • Flexible images
    • Media queries
  • Responsive design strategies:
    • Desktop-First
    • Mobile-First
    • Everything-in-Parallel
  • Content elements on web pages, such as images and charts (visualizations), also have to become responsive.
